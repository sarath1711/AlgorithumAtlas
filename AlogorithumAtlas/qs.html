Quick Sort

def partition(array, low, high):
    pivot = array[high]
    i = low - 1
    for j in range(low, high):
        if array[j] <= pivot:
            i = i + 1
            (array[i], array[j]) = (array[j], array[i])

    (array[i + 1], array[high]) = (array[high], array[i + 1])

    return i + 1

def quicksort(array, low, high):
    if low < high:
        pi = partition(array, low, high)

        quicksort(array, low, pi - 1)
        quicksort(array, pi + 1, high)

EXPLANTION

Quick Sort Algorithm

- The code defines two functions: `partition` and `quicksort` to implement the Quick Sort algorithm.
- Quick Sort is a divide-and-conquer algorithm that works by selecting a 'pivot' element from the array and partitioning the other elements into two sub-arrays according to whether they are less than or greater than the pivot.
- The `partition` function takes an array, a low index, and a high index as input parameters.
- It selects the last element of the array as the pivot.
- It then iterates through the array from the low index to the high index (excluding the pivot).
- If an element is less than or equal to the pivot, it increments the `i` variable and swaps the element at index `i` with the element at index `j`.
- Finally, it places the pivot element in its correct position by swapping it with the element at index `i + 1`.
- The function returns the index of the pivot element after partitioning.

Quicksort Function

- The `quicksort` function takes an array, a low index, and a high index as input parameters.
- It recursively sorts the sub-arrays before and after the pivot element.
- If the low index is less than the high index, it calls the `partition` function to get the index of the pivot element.
- It then recursively calls `quicksort` on the sub-array to the left of the pivot (elements less than the pivot) and the sub-array to the right of the pivot (elements greater than the pivot).

Main Function

- The main function reads input from the user to create an array.
- It then calls the `quicksort` function to sort the array in ascending order.
- Finally, it prints the sorted array.
