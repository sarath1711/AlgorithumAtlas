Naïve String Matching

Code

pattern = input("Enter the pattern:")
text=input("Enter the text: ")
n = len(text)
m = len(pattern)
for i in range(n - m + 1):
  j = 0
  while j < m and text[i + j] == pattern[j]:
      j += 1
      if  j == m:
          print(f"patter is found at {i}")

EXPLANTION

 Naïve String Matching is a simple algorithm used to find all occurrences of a pattern in a text. Here are the key points:

 Algorithm Overview: Naïve String Matching, also known as Brute-Force String Matching, compares the pattern with all possible positions in the text.
            
 Step-by-Step Process:
            
 Start at the first character of the text and slide the pattern over it.
 Compare each character of the pattern with the corresponding character in the text.
 If a mismatch occurs, shift the pattern one position to the right in the text and continue comparing.
 Repeat this process until the entire text is processed or a match is found.
 Complexity: The worst-case time complexity of Naïve String Matching is O((n-m+1)m), where n is the length of the text, and m is the length of the pattern. It is not the most efficient string matching algorithm,
 especially for large texts and patterns.
            
 Use Cases: Naïve String Matching is useful when simplicity is preferred over efficiency. It is often used for educational purposes and as a starting point for understanding more advanced string 
 matching algorithms.
            
 Drawbacks: This algorithm can be slow for long texts and patterns due to its high time complexity. More advanced algorithms like KMP and Boyer-Moore offer better performance in practice.
